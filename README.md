# JPART-replication
# Data and Code for JPART Submission

This repository contains the data and Stata code used for the analysis in the manuscript submitted to *JPART*.

## Files:
- `Master.do`: Stata do-file containing the code for data processing and analysis. This script includes all the necessary steps to replicate the results presented in the paper.
- `main.dta`: The main dataset used in the analysis. It includes all the variables needed to run the models and conduct the statistical tests discussed in the manuscript.

## Instructions:
1. Download both the `Master.do` and `main.dta` files.
2. Open Stata and run the `Master.do` file to replicate the analysis.
3. Ensure that the working directory in Stata is set to the folder where the files are saved.
4. If you have any issues or questions about the data or code, please feel free to contact me.
